# 📝 알고리즘 문제 풀이 - 2025-03-27

## 문제 📖

- **문제 링크:** [백준 - 내려가기](https://www.acmicpc.net/problem/2096)

- **문제 유형:** 다이나믹 프로그래밍(Dynamic Programming)

- **난이도:** Gold.5

- **풀이 시간:** ⏳ 1시간 30분 - 실제 걸렸던 시간

## ✍ 접근 방식

- DP 업데이트:
  - 두 번째 행부터 마지막 행까지 반복하며, 각 행의 3개의 숫자(a, b, c)에 대해
  - 인덱스 0: 이전 행의 인덱스 0, 1 중 더 큰 값/더 작은 값을 선택하여 a와 더하기.
  - 인덱스 1: 이전 행의 0, 1, 2에서 최대/최소 값을 선택하여 b와 더하기.
  - 인덱스 2: 이전 행의 1, 2에서 최대/최소 값을 선택하여 c와 더하기.
  - 각 행마다 새로운 dp 배열(newMax, newMin)을 계산 후 기존 dp 배열을 업데이트.
  - 이 방식은 각 행마다 6개의 정수만 저장하므로, N이 커도 공간 복잡도는 O(1).
- 결과 도출:
  - 마지막 행까지 DP 업데이트를 완료한 후, dpMax 배열에서 최댓값, dpMin 배열에서 최솟값을 구해 출력.
