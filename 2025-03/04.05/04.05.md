# 📝 알고리즘 문제 풀이 - 2025-04-05

## 문제 📖

- **문제 링크:** [백준 - 양치기 꿍](https://www.acmicpc.net/problem/3187)

- **문제 유형:** BFS

- **난이도:** 실버1

- **풀이 시간:** ⏳ 8분

## ✍ 접근 방식

- 1. BFS를 사용하여 그래프를 탐색합니다.
- 2. 각 영역에서 양과 늑대의 수를 세어 비교합니다.
- 3. 양이 더 많으면 양의 수를 증가시키고, 늑대가 더 많으면 늑대의 수를 증가시킵니다.
- 4. 최종적으로 양과 늑대의 수를 출력합니다.
