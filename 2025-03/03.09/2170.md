# 📝 알고리즘 문제 풀이 - 2025-03-05

## 문제 📖

- **문제 링크:** [백준 - 선 긋기 / 2170](https://www.acmicpc.net/problem/2170)

- **문제 유형:** 그리디

- **난이도:** Gold 5

- **풀이 시간:** 1시간 10분 (시간초과 때문에 고생..)

## ✍ 접근 방식

1. 완전히 겹치는 구역, 조금 겹치는 구역, 그렇지 않은 구역을 나눈다.

- 완전히 겹치는 구역: start,end 변경 x
- 조금 겹치는 구역: end 변경
- 그렇지 않은 구역: 새로운 영역이므로 answer에 길이를 더해준다.
