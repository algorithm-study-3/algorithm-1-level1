## ✍ 접근 방식

풀이 1
- DP 사용
- 0인 부분은 0, 1인 부분은 Infinity 값을 가지는 DP Matrix를 샹성한다.
- 이중 for문으로 DP Matrix 값이 0 이면 패스, 아니면 상하좌우를 확인해서 최소인 값의 + 1의 값으로 수정한다.
- 이중 for문을 반대로 한번 더 실행한다.

풀이 2
- BFS 사용
- 0인 부분을 queue에 넣는다.
- queue에 있는 값을 하나씩 꺼내보면서 상하좌우에 +1 값을 적용한다.
- 그리고 상하좌우를 queue에 넣는다.
- queue가 빌 때까지 반복
