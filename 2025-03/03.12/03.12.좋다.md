# 📝 알고리즘 문제 풀이 - 2025-03-12

## 문제 📖

- **문제 링크:** [백준 - 좋다](https://www.acmicpc.net/problem/1253)

- **문제 유형:** 투포인터

- **난이도:** 골드4

- **풀이 시간:** ⏳ 20분 (다시풀기 1회차)

## ✍ 접근 방식

- 투 포인터 문제
- 오름차순 정렬 먼저 해야함.
- 두 수의 합이 다른 수가 되는지 확인하므로 start 와 end가 만나면 안됨.
- 찾는 수가 다른 두 수의 합이 되는지 확인하므로 다음과 같이 수를 나눔.
  - 찾는 수: target, 나머지 수: rest
