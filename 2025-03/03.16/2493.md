# 📝 알고리즘 문제 풀이 - 2025-03-05

## 문제 📖

- **문제 링크:** [백준 - 탑 / 2493](https://www.acmicpc.net/problem/9012)

- **문제 유형:** 스택

- **난이도:** Gold 5

- **풀이 시간:** 30분

## ✍ 접근 방식

1. 왼쪽에서 오른쪽으로 탑을 확인하며 신호 수신 여부를 판단
2. 스택에 (인덱스, 높이) 저장 후 현재 탑과 비교
3. 현재 탑보다 낮은 탑은 스택에서 제거
4. 더 높은 탑이 나오면 인덱스 저장 후 종료
5. 현재 탑을 스택에 추가
